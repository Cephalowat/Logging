# Wait-Logging

## SYNOPSIS
Wait for the message queue to be emptied

## SYNTAX

```
Wait-Logging [<CommonParameters>]
```

## DESCRIPTION
This function can be used to block the execution of a script waiting for the message queue to be emptied

## EXAMPLES

### EXAMPLE 1
```
Wait-Logging
```
